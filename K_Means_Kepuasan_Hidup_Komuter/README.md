# K-Means: Kepuasan Hidup Komuter
### Pengelompokan Komuter Berdasarkan Indikator Kepuasan Hidup dengan K-Means Clustering

Repositori ini memuat syntax R (dalam bentuk makrdown) beserta interpretasi dan analisis serta dilengkapi dengan dataset yang digunakan mengenai pengelompokan komputer berdasarkan kepuasan hidupnya dengan metode K-Means Clustering. Syntex tersebut diselesaikan pada Desember 2021 sebagai jawaban ujian akhir semester mata kuliah Data Mining, semester 7, Politeknik Statistika STIS. Semoga bermanfaat.
